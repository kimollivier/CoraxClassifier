Installation

 

 

 

Installation Requirements

 

Hardware

A personal computer that can run QGIS. It can be Windows, Linux or Apple.

Maps and QGIS are very visual. I recommend using a computer mouse in preference to a trackpad.

 

QGIS

Install QGIS version 3.40.12-Bratislava long term release or other 3.4x versions.

 

Computer Vision

To view the images and videos we need some external applications. There will be some default applications installed that will open if the file types are recognised. These may not be as useful as some other programs.

 

Optionally install OpenCV2 to enable first frame preview of videos.

From the OSGeo4w shell: python -m pip install opencv-python-headless

 

The Classifier APP can preview still images but an extra python package called CV2 is required to preview the first frame of each videos If CV2 is not installed then video previews will be disabled.

 

I prefer to view still images using IrfanView and videos using VLC. Both of these are free and are very capable of editing the images and videos.

 

CoraxClassifier Plugin

Download the Corax Classifier zipped package. This is a QGIS plugin.

It needs a camera sites layer, catalog template and species lookup table.

 

https://github.com/kimollivier/CoraxClassifier

 

Create a QGIS project and add the required layers. They must be called image_classification and camera_loc because they are hard coded. This may change, the template may be built-in. For flexibility the camera_loc and bird_pest_lut will be external to the plugin, stored in a geopackage. A sample package with all the files and a project can be downloaded.

 

The plugin is a zip file that can be downloaded.

 

A sample project file, templates in a geopackage can be downloaded

 

You should have a folder structure like this:

 

 

•This project is called BandedRailMonitoring (I avoid spaces in all folder and file names)

•Each reserve has a folder

•Within each reserve there are folder for each downloaded memory card which must start with the camera-id and date exactly like that.

•There are two geopackages (*.gpkg), one for the camera locations and another for the classifier templates. The camera locations must be points in the classifier.gpkg

•There is a QGIS project (*.qgz) that will run the classifier

•The CoraxClassifier plugin to be loaded is the zip file (old name in image)
