Reporting

 

Reporting is assumed to use QGIS tools, or the database could be uploaded to ArcGISOnline to be used in a Dashboard.

Maps are limted because the camera tends to be static so there are a lot of photos in one place.

 

A pie chart can be placed on each point showing the distribution of species.

 

A chart can be created with summary counts of species over time.
