TrapNZ Camera Monitoring Stations

 

The first two stations are in Kauri Point Halo which is a good way to record the location. Camera Bandits will need to be added to this Project and any others where the traps are placed. It is not proposed to create a separate project just for cameras.

 

 

You need to be added to the project to be able to add new camera sites. Although it is possible to upload and classify images, the classification sets are limited, it does not allow videos and the images are too small to inspect properly. The AI classification is weak.
