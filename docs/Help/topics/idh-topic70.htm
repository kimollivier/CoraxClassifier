Operation

 

Find the project folder and upload the folder from the memory card into the data folder.

 

Rename the new folder with the camera id and date. It must be like this: CAM99-2025-11-17 where 99 is the unique serial number based on the camera location, not the camera. If it is a new location it should have been captured in TrapNZ, otherwise edit the camera_loc layer and add it manually.

 

Open the QGIS project corax_classifier.qgz

 

It should have a map of the project area with the current camera locations labelled. You should see your camera.

 

 

Install the Plugin

 

plugins>Install from Zip

 

 

 

Run the autofill_table

 

Open the plugin from the toolbar Corax icon:

 

Open the Image Loader

 

 

The loader tool will prompt you to locate the new image folder eg CAMnn-2025-11-01

You will be prompted for a new table name eg CAMnn (any unique name)

It will add this table to the QGIS table of contents and report the number of images and videos.

If this is a following card for the same camera site use the same name and it will append the catalog records

 

Open the table as a form

 

Select the new table at the top of the Classifier

 

 

Step through the records and classify by opening the image or video and choosing the species from the dropdown.

You can step backward and change any setting. You can add a comment, a second species and the count if more than one.

The edits are saved each time you step to another record. The save is for the last record edited.

 

You can edit during the slideshow. Currently set to 2 seconds. A speed dial is coming.

 

Optionally save the QGIS Project to be able to easily reopen the editor.

 

Each card of images should be stored in the same metadata table name. You may want to separate still and video images.
