AI Species Recognition

This is unlikely to be possible for a long time. Why?

•Image recognition requires training on hundreds of example tagged images. We don't have hundreds of images of each rare bird we want to recognise, let alone all the common birds.

•As you can trial in the TrapNZ recognition service the error rate is very high and it only recognises a 'bird'!

•Even if we had a better trained AI model we would need considerable processing power and a GPU to run the inferencer. Most people do not have the processing power and it is expensive to use a cloud based service.

•There is more to collect than just recognising the species. What activity is in the video? How many animals? Notes need adding.
